# simple-etl-pipeline
This project demonstrates a basic ETL (Extract, Transform, Load) pipeline built using Python. The pipeline extracts sales data from a CSV file, processes it using Pandas, and loads the data into a SQLite database.

# Simple ETL Pipeline

This project demonstrates a basic ETL pipeline.

## Process
1. Extract data from CSV
2. Transform data using Pandas
3. Load data into SQLite database

## Tools
Python
Pandas
SQLite
